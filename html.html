O que é HTML?

HTMLé a linguagem de marcação usada para criar
e estruturar páginas da web. Ele fornece a estrutura básica para o conteúdo de uma
página, incluindo texto, imagens, links e outros elementos.

HTML é uma linguagem fundamental para o desenvolvimento web e é frequentemente
combinada com outras tecnologias, como CSS (Cascading Style Sheets) para
estilizar a página e JavaScript para adicionar interatividade e funcionalidade. Juntos,
esses três componentes formam a base da maioria das páginas da web modernas.

O que são Tags?

Em HTML, os desenvolvedores usam tags para definir diferentes tipos de conteúdo
e sua formatação na página. Por exemplo, a tag <p> é usada para indicar parágrafos
de texto, a tag <img> é usada para inserir imagens, e a tag <a> é usada para criar links.
As tags são combinadas com atributos para fornecer informações adicionais sobre
como o conteúdo deve ser exibido ou comportar-se.

Algumas tags:

  • <html>: Define o início e fim de um arquivo HTML
  • <head>: Usada para colocar as configurações, tudo que será carregado antes de exibir a página.
  • <title>: Usada para definir o texto que aparece na aba do navegador
  • <body>: Usada para definir o conteúdo que aparecerá para o usuário
  • <p>: Usada para indicar paragráfos de texto
  • <strong>: Usada para deixar um texto em negrito.
  • <i>: Usada para colocar um texto em itálico
  • <img>: Usada para incluir imagens na página
  • <input>: Usada para incluir campos
  • <a>: Usada para incluir um link dentro da página
  • <h1>,<h2>,<h3>,<h4>,<h5>,<h6>: Usadas para definir títulos no texto, sendo <h1> o mais importante e <h6> e menos importante
  • <blockquote>: Usada para incluir

  • <ul>: Usada para criar listas não ordenadas

    • O que são atributos?

    Em HTML, um atributo é uma parte adicional de uma tag que fornece informações
    extras sobre o elemento. Os atributos são usados para modificar ou fornecer detalhes
    específicos sobre como um elemento deve ser exibido ou comportar-se.

    Por exemplo, na tag de imagem <img>, o atributo "src" especifica o caminho do
    arquivo da imagem que deve ser exibida. Outros atributos comuns incluem "alt"
    para texto alternativo, "href" para links em tags <a>, "class" e "id" para atribuir
    classes e identificadores CSS para estilos e seletores JavaScript, e muitos outros.

    Os atributos são sempre fornecidos na forma de pares chave-valor, separados por
    um sinal de igual ("="). Por exemplo:

    <img src="exemplo.jpg" alt="Exemplo de imagem">

    Neste exemplo, "src" é o nome do atributo e "exemplo.jpg" é o valor atribuído a
    ele. Os atributos são essenciais para personalizar e controlar o comportamento dos
    elementos HTML em uma página da web.

    Alguns atributos mais comuns:

    • id: Define o identificador único para cada tag
    • style: Usado para configurar o estilo das tags
    • class: Usado para atribuir uma classe a uma tag, permitindo estilizar múltiplas
      tags ao mesmo tempo
    • type: Usado para definir o tipo de algumas tags, como a tag <input>
    • src: Usada para definir o endereço do arquivo de imagem para a tag <img>
    • width: Serve para definir a largura de um elemento
    • height: Serve para definir a altura de um elemento
